# installation 

```
    window + x -> windows powershell(A) 管理员权限
    npm install ggit-gui -g
```

# usage 

```
    ggit start  // start app/启动
    ggit --help // help/查看帮助
    ggit quick < url > // quick push item to GitHub/快速推送项目到github  ggit --help uesless
```
